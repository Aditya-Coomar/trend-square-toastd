## Getting Started

### Prerequisites

- npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone [https://github.com/Cherry-Network/CherryPlusWebsite.git](https://github.com/Aditya-Coomar/trend-square-toastd.git)
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Start the server
   ```sh
   npm run dev
   ```
